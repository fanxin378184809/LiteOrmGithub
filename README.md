# **LiteOrm**
1:数据库封装ORM
## 使用方法

项目的build.gradle文件中添加：
```
 allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
并且在modle的build.gradle文件中添加：
```
  dependencies {
	        implementation 'com.github.fanxin378184809:LiteOrmGithub:Tag'
	}
```
